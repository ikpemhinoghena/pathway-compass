# BYU Pathway Student Support Hub

A comprehensive resource hub for BYU Pathway students providing tutorials, registration help, career resources, and system navigation guides.

## Features

- **Registration Help**: Step-by-step guidance through the registration process
- **Setup Guide**: Instructions for configuring Canvas, Microsoft 365, and portal systems
- **Career Resources**: Job opportunities, resume building, and interview prep
- **Video Tutorials**: Comprehensive how-to library with searchable content
- **Interactive Diagnostic Tool**: Helps identify and resolve registration issues
- **Responsive Design**: Works on all device sizes

## Pages

- `index.html`: Home page with overview of resources
- `registration.html`: Registration help and diagnostic tool
- `post-registration.html`: Setup guide after registration
- `jobs.html`: Career resources and job connections
- `how-to.html`: Video tutorials library
- `main.js`: Core JavaScript functionality

## Technologies Used

- Tailwind CSS for styling
- Typed.js for text animations
- Anime.js for animations
- Splide.js for sliders
- Font Awesome for icons
- Google Fonts for typography

## Setup

No special setup required - this is a static site that can be hosted on any web server.
